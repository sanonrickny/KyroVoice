# KyroVoice 🎙️

Local-first voice dictation for macOS. Powered by Whisper on Apple Silicon.

## ✨ Features
- **100% Private**: All processing happens on-device.
- **Smart Modes**: Normal, Email, and Code-optimized dictation.
- **Seamless**: Hold a hotkey, speak, and text appears at your cursor.

## 🚀 Quick Start

### Requirements
- macOS 13+ (Apple Silicon)
- Xcode 15+

### Build & Run
```bash
./build.sh
./run.sh
```

## 🛠️ Usage
- **Hotkey**: `⌘ ⇧ Space` (Hold to talk)
- **Menu Bar**: Control modes, Whisper models, and settings.
